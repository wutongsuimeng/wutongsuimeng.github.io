修复进入grub rescue的问题

2021-03-21 17:37:56
linux
约 193 字 预计阅读 1 分钟 次阅读

文章目录

重装window后，突然就进去不了linux系统，在命令行窗口下提示

error :file '/boot/grub/x86_64-efi/normal.mod' not found

解决办法:

# 列出所有磁盘
> ls
> (hd0,gpt1) (hd0,gpt2)
# 一个一个查看磁盘里面的内容,找到grub所在位置,如果有提示/boot,/grub,即很有可能便是grub的位置
> ls (hd0,gpt2)/
> ./ ../ boot/ dev/ 
# 设置为启动分区
> set prefix=(hd0,gpt2)/boot/grub
> set root=hd0,gpt5
> insmod normal
> normal

之后便会重启系统

进入到系统之后,重新安装grub

# 查看/boot/efi的挂载点
> df
# 安装grub
> sudo grub-install /dev/sda1
