实现串流的软件有多种：

Integer中bitCount方法的源码解析

2023-12-10 17:13:03
约 912 字 预计阅读 2 分钟

Integer 中的 bitCount() 方法用于统计二进制数字中 bit 为 1 的个数，例如 0b111 中 bit 为 1 的个数有 3 位。

源码为：

public static int bitCount(int i) {  
    // HD, Figure 5-2  
    i = i - ((i >>> 1) & 0x55555555);  
    i = (i & 0x33333333) + ((i >>> 2) & 0x33333333);  
    i = (i + (i >>> 4)) & 0x0f0f0f0f;  
    i = i + (i >>> 8);  
    i = i + (i >>> 16);  
    return i & 0x3f;  
}

WSL登录时提示nsenter: cannot open /proc/320/ns/time: No such file or directory的解决办法

2023-11-23 22:40:42
约 100 字 预计阅读 1 分钟

在登录 WSL 的 Ubuntu 时，不仅要求 root 权限，还登录失败，提示“nsenter: cannot open /proc/320/ns/time: No such file or directory”。

解决办法是在 powershell 中执行 “wsl – sudo vi /etc/profile”命令，删除文件内容，保存便可成功登录。
